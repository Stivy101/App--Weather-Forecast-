WeatherSc - небольшой скрипт, который предоставляет информацию о погоде на текущий день с помощью обращения по API к одному из сайтов синоптиков
Минималистичный графический интерфейс создан при помощи библиотеки PySimpleGUI
Скрипт был запакован в exe при помощи библиотки Pyinstaller

Код предусматривает следующие проверки:
- ввод городов регистронезависим, скрипт сам превращает написанное в нужную форму
- если сервер не находит информации по городу, выводится соответствующиее сообщение

Использованные при написании библиотеки и модули:
- requests
- PySimpleGUI
- PyInstaller
